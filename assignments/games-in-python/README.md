# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic Hangman word-guessing game using Python. In this assignment, you will practice string manipulation, loops, conditionals, random word selection, and user input handling.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Create a command-line Hangman game where the player guesses one letter at a time to reveal a hidden word before running out of attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list of possible words.
- Accept single-letter guesses from the user and update game progress in `_ _ _` format.
- Track and display the number of incorrect guesses remaining.
- End the game when the full word is guessed or when attempts are exhausted.
- Display a clear win message for success and a clear lose message that shows the correct word.
